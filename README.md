## Getting data

- Order (Ascending order and descending order)
- Limit (limit the amount of record returned)
- Group by
- Having
- Function (aggregate function and scalar function

### Select
从表中获取栏, 如获取所有的栏。

```
SELECT *
FROM table_name
```

获取特定的栏。
```
SELECT column_name
FROM table_name
```
